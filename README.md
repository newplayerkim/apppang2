# 앱팡 (App-Pang)

## 시작하기

```bash
git clone https://github.com/inu-appcenter/basic-project-apppang.git
cd basic-project-apppang
npm install
npm run dev
```


---

## 프로젝트 소개

**앱팡**은 모바일 환경에 최적화된 이커머스 쇼핑 앱 프론트엔드입니다.

상품 탐색부터 장바구니, 주문, AI 쇼핑 챗봇까지 쇼핑에 필요한 핵심 기능을 제공합니다.

### 주요 기능

| 기능            | 설명                                      |
| --------------- | ----------------------------------------- |
| 메인            | 배너 슬라이더, 카테고리 아이콘 메뉴       |
| 검색            | 자동완성, 최근 검색어, 추천 검색어        |
| 상품 리스트     | 무한 스크롤, 정렬, 카드형 리스트          |
| 상품 상세       | 이미지 슬라이더, 옵션 선택, 리뷰, 찜하기  |
| 챗봇            | AI 쇼핑 도우미, 상품 카드 응답, 빠른 답변 |
| 장바구니        | 수량 조절, 선택 삭제, 결제 금액 요약      |
| 마이페이지      | 주문 내역, 위시리스트, 최근 본 상품       |
| 로그인/회원가입 | JWT 기반 인증, 자동 로그인                |

### 기술 스택

- **Framework**: React 19 + TypeScript
- **Build Tool**: Vite
- **Styling**: TailwindCSS v4
- **상태 관리**: Zustand
- **HTTP 클라이언트**: Axios
- **라우팅**: React Router v7
